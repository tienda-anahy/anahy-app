🍍 Configura tu panel Anahy

Para que los datos se guarden y se compartan entre todos los móviles, necesitas conectar una base de datos gratuita de Firebase. Son 5 minutos, una sola vez:

  1. Ve a console.firebase.google.com e inicia sesión con tu Google.
  2. Crea un proyecto nuevo (el nombre no importa, ej. "anahy-tienda").
  3. En el menú lateral, entra en Firestore Database → "Crear base de datos" → modo de prueba (test mode) → elige una región cercana (europe-west).
  4. Ve a Configuración del proyecto (el icono de engranaje) → en "Tus apps" pulsa el icono </> (Web) → dale un nombre → "Registrar app".
  5. Te aparecerá un bloque de código con firebaseConfig = {"{"} apiKey: ..., projectId: ..., {"}"}. Copia esos valores.
  6. Abre el archivo index.html con cualquier editor de texto (o directamente en GitHub), busca la palabra CONFIGURA_AQUI y sustituye ese objeto por el que has copiado.
  7. Guarda, sube los cambios a GitHub, y recarga esta página.

Mientras tanto, aquí tienes cómo debería quedar (ejemplo, no lo copies literal):

const firebaseConfig = {
  apiKey: "AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXX",
  authDomain: "anahy-tienda.firebaseapp.com",
  projectId: "anahy-tienda",
  storageBucket: "anahy-tienda.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc123"
};

Nota de seguridad: en modo de prueba, Firestore queda abierto 30 días a cualquiera que tenga tu configuración. Para un negocio familiar pequeño el riesgo es bajo, pero si quieres más adelante te ayudo a poner reglas de acceso más estrictas.
